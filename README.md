## How to Use It

1. Clone This Repo:
```bash
git clone https://github.com/rakhaafd/shell.git
```

2. Go to the Project Directory:
```bash
cd project
cd shellearn
```

3. Give Permission to .sh file:
```bash
chmod +x xxxxx.sh
```

4. Run .sh file:
```bash
./xxxxx.sh
```
